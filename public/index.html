And When You're Done, Host it!

1. Create / Sign into your GitHub account

Click here to navigate to Github

2. Make a Repository

Go into your dashboard and find a “New Repository” button.

Click on it to create a new repository.

3. Upload The Files From the Folder, "portfolio"

Click "Upload an existing file" highlighted in blue.

Then drag and drop every file inside the portfolio folder.

Click "commit changes".

4. Navigate to Setting

Click "Upload an existing file" highlighted in blue.

Then drag and drop every file inside the portfolio folder

Click "commit changes".

5. Navigate to Pages

Select the "pages" tab on the left hand side.

6. Build the Page

Select "main" branch.

Click "save".

7. Congrats

Your github link to the portfolio will be

https://username.github.io/portfolio-name
